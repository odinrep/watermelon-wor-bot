# Watermelon WOR Bot

Telegram bot for WOR range tracking, journaling, and Google Sheets sync.

## Commands
- /wor help
- /wor finalize [YYYY-MM-DD]
- /wor enter <YYYY-MM-DD> <long|short> <size%> (risk-based)
- /wor exit <YYYY-MM-DD> <win|loss> [pnl_r?]
- /wor equity
- /wor twr [usd|index]
- /sheets ping

## Deploy
- Railway: Deploy from GitHub, set env vars from `.env.example`.
